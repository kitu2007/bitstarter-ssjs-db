Isn't it waste to have sprinklers on when it rained an hour ago. Wish you had a way to tell the sprinklers not to start if it rained not long ago. Now you can have a system which can do this and lot more. It can tell you if it rained and how heavily it rained, so you can turn the sprinklers on when needed. And not just that with this new instrument you can be warned about when it starts raining. The instruments can collect data on rain where it matters most, in your backyard or farm.

The instrument works by using a simple camera and a proprietary algorithm to detect if it is raining and to get an estimate of the rain rate. It can collect data with high temporal accuracy. The instrument is connected to the sprinkler system and can decide if the sprinklers need to be turned on or if the rain was light how much more water is needed.
